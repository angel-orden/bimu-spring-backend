# BIMU Spring Backend

Backend REST API for BIMU, a cycling route management platform.

This project is a Spring Boot backend built to demonstrate backend development skills with Java, REST APIs, authentication, PostgreSQL and clean architecture principles.

## Tech Stack

- Java 21
- Spring Boot
- Spring Web
- Spring Security
- JWT Authentication
- Spring Data JPA
- PostgreSQL
- Docker
- Swagger / OpenAPI
- JUnit

## Core Features

- User registration and login
- JWT-based authentication
- Role-based access control
- Route CRUD
- Join cycling outings
- Protected REST endpoints
- PostgreSQL persistence
- API documentation with Swagger

## Project Goal

The goal of this project is to rebuild the backend of BIMU using a more enterprise-oriented Java stack, focusing on maintainability, security and scalability.
